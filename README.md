Program to organise Carpentries Workshops

## Getting Started

### Prerequisites

Java : openjdk 11.0.17 2022-10-18 (or equivalent)

### Installation

This is a maven application and should build with:
`mvn package`

### Running Locally

`java -cp workshop_admin.jar ac.uk.ncl.workshop_admin.Main`
